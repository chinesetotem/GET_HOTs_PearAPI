# 热点新闻聚合器

这个项目使用 GitHub Actions 自动获取各大平台的热点新闻，并将数据保存到仓库中。

## 功能特点

- 自动获取12个主要平台的热点新闻
- 每4小时自动更新一次
- 支持手动触发更新
- 数据以JSON格式存储，按时间戳命名

## 支持的平台

1. 哔哩哔哩
2. 百度
3. 知乎
4. 百度贴吧
5. 少数派
6. IT之家
7. 澎湃新闻
8. 今日头条
9. 微博热搜
10. 36氪
11. 稀土掘金
12. 腾讯新闻

## 数据格式

每个平台的数据包含以下字段：
- `title`: 热点标题
- `desc`: 热点描述

## 自动化运行

项目使用 GitHub Actions 实现自动化运行：
- 定时任务：每4小时自动运行一次
- 手动触发：可以通过 GitHub Actions 页面手动触发运行

## 数据存储

- 所有数据存储在 `data` 目录下
- 文件名格式：`hot_news_YYYYMMDD_HHMMSS.json`
- 使用 UTF-8 编码存储，支持中文
